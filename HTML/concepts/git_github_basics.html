GIT one by one commands:
  • git init (asking git to keep the track)
  • git add (adding files and telling git to keep track on which file)
  • unmodified/modified (if the file has changed or not if not then in unmodified state and if changed then in modified state)
  • staged (when we are commiting after changes)
  • commit sirf stagging area se hi ho skta hai.. to hamesha add krna padega taaki staging area mei laaya sake to commit
  • git log :- to see previous commit history
